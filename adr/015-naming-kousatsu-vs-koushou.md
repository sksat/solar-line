# ADR-015: プロジェクト名称 — 考察 vs 考証

## Status

Accepted (2026-02-25)

プロジェクトオーナーにより選択肢 A（「考証」への変更）が承認された。

## Context

プロジェクトオーナーから「これ、考察というよりは考証か？」という指摘があった。

現在のプロジェクト名は「SOLAR LINE 考察」だが、実際のプロジェクト内容は：
- 軌道力学の計算に基づく**数値検証**（物理法則との照合）
- パラメータ空間の**探索**（質量・推力・Isp の境界条件特定）
- 複数エピソード間の**整合性検証**（船体スペック、タイムライン）
- 作品の物語的整合性の**議論**（ナラティブ妥当性）

前者 3 つは「考証」（文献・証拠に基づく客観的検証）の性質が強く、最後の 1 つは「考察」（主観的な分析・推測）に近い。

### 用語の比較

| 用語 | 意味 | ニュアンス | 用例 |
|------|------|-----------|------|
| 考察（kousatsu）| 物事を明らかにするために深く考えること | 個人の考え・推測を含む、主観的な分析 | アニメ考察、物語考察 |
| 考証（koushou）| 文献・証拠に基づいて事実を検証すること | 客観的・学術的な裏付け調査 | 時代考証、科学考証 |

## Decision

3 つの選択肢を提案する。オーナーの判断を待つ。

### 選択肢 A: 「SOLAR LINE 考証」に変更（推奨）

プロジェクトの主要活動（軌道計算、パラメータ検証、数値比較）は「考証」の性質が強い。
「科学考証」という既存の用語との親和性も高い。

**影響:**
- サイトタイトル変更
- GitHub Pages URL は変わらない（リポジトリ名依存）
- CLAUDE.md、README、報告書内の表記更新が必要

### 選択肢 B: 使い分け

- 数値検証セクション → 「考証」
- 総合的な議論・推測 → 「考察」
- サイトタイトルは「SOLAR LINE 考証」（主要活動に合わせる）

### 選択肢 C: 現状維持（「考察」のまま）

「考察」はアニメ分析コミュニティで広く使われており、読者にとって馴染みがある。
変更のコストに見合わない可能性もある。

## Alternatives Considered

上記 3 選択肢の通り。選択肢 A を推奨するが、C の「読者の馴染み」も重要な観点。

## Assumptions

- プロジェクトの主要価値は数値検証（考証）にある
- 読者はアニメファンが中心で「考察」の方が馴染みがある可能性がある
- URL（リポジトリ名）の変更は不要

## Consequences

**選択肢 A の場合:**
- メリット: プロジェクトの性質をより正確に表現
- デメリット: 表記更新の手間、読者にとっての馴染みが薄い
- 影響: サイトタイトル、ナビゲーション、レポート本文の更新が必要
