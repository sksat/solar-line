# Task 107: 科学的精度レポートのスコープ再構成

## Status: DONE

## Motivation

人間の指示: 「『精度』という観点でひとつのレポートを書くとスコープと内容がぼやけているように見える。また、現実の観測値などと単なるパラメータが同等であることは精度の問題ではなく考証の出典とその確からしさの問題。」

現在の `science-accuracy.json` は以下を混在させている:
- 現実の物理法則との整合性（光速遅延、ΔV計算）→ 精度の問題
- 設定資料のパラメータと作中描写の一致 → 考証・出典の問題
- 物語内の整合性（クロスエピソード一貫性）→ 別の問題

## Scope

1. `science-accuracy.json` を廃止し、内容を以下に再配分:
   - 物理法則検証 → 各話分析の個別遷移カードに統合（既にある程度できている）
   - パラメータ出典管理 → `cross-episode.json` の考証セクションに移行
   - クロスエピソード整合性 → `cross-episode.json` に統合
2. 「精度」ではなく「考証」（考察と典拠）の観点でまとめ直す
3. 各パラメータの出典の確からしさを明示（一次資料/二次資料/推定の区別）
4. ナビゲーション更新

## Dependencies

- 各話分析（006, 008, 015, 020, 023）— すべて DONE
- cross-episode.json — DONE
