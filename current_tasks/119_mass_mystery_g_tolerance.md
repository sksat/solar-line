# Task 119: 質量の謎を耐G SF設定仮説で再考

## Status: TODO

## Description
ケストレル号の質量の謎（公称48,000tだが計算から導かれる値との不一致）について、
「耐G方面にSF的な設定がある」という仮説で再分析する。

例えば、乗組員の耐G能力が現代人より大幅に高い（慣性ダンパー等の技術がある）場合、
許容加速度が変わり、質量・推力の関係式が変わる。

## Approach
1. 現在の質量の謎の整理（ship-kestrel.json の分析を参照）
2. 耐G SF設定の仮説を複数立てる（慣性ダンパー、薬物、生体改造等）
3. 各仮説で許容加速度を変えた場合の質量計算
4. どの仮説なら整合するか分析
5. ship-kestrel.json レポートの更新

## Dependencies
- reports/data/summary/ship-kestrel.json（既存分析）
- 各エピソードの加速度データ

## Origin
人間指示: 「質量の謎は、耐Gの方に SF 的な設定があるのものだと仮定してみて考え直してみてほしい」
