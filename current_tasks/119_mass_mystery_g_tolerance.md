# Task 119: 質量の謎を耐G SF設定仮説で再考

## Status: DONE

## Description
ケストレル号の質量の謎（公称48,000tだが計算から導かれる値との不一致）について、
「耐G方面にSF的な設定がある」という仮説で再分析する。

## Result
4つの仮説を検討し、ship-kestrel.json に「耐G仮説: SF的設定からの質量の謎の再考」セクションを追加:

- 仮説A: 慣性補償技術（慣性ダンパー） — **最も整合**
  - 実加速度3.3Gだが慣性補償で体感0.2G以下
  - きりたんの1G描写（第3話）との矛盾なし
- 仮説B: 耐G技術で2-3G持続 — きりたんの1G描写と矛盾
- 仮説C: 推進剤消費による可変質量 — 出発時に不整合
- 仮説D: 48,000tはDWT（載貨重量トン数） — **有力**

**結論: 仮説A+D（慣性補償 + DWT解釈）が最も全描写と整合する。**

Key evidence:
- きりたん「1Gは重すぎる」(EP03 6:24) → 0.8G適応者が3Gに耐えるには補償必須
- ケイ「減速ショック28%心停止」(EP02 9:40) → 補償が不完全な場面もある
- 骨格フレーム応力114% (EP02) → 船体側も限界ギリギリ

補償率別の体感Gテーブルを含むデータ表も追加。
