# Task 108: 外部エージェントによるレポートレビュー体制

## Status: DONE

## Motivation

人間の指示: 「レポートのレビューは、gpt-5.2 や Task agent などの別コンテキストを持ったエージェントにやらせること。その際は、SOLAR LINE に対する事前知識の無い SF ないし軌道力学に関心があるが詳しくはないペルソナをシステムプロンプトで設定して読ませるとよい。ドラフトのレビューとは別に、最終的なレビューは一時ディレクトリで起動することでコードベースなどを見せない状態で、デプロイされているページにアクセスさせることで行うとよい。」

## Scope

### ドラフトレビュー（コードベースあり）
1. `report-review` スキルを更新:
   - レビュアーペルソナのシステムプロンプトを設定
     - 「SF作品と軌道力学に関心はあるが詳しくはない読者」
     - SOLAR LINE の事前知識なし
   - gpt-5.2 への物理コンサルテーション機能を追加（推論モデル向け）
   - Task agent (Sonnet/Haiku) でのレビューも並列実行

### 最終レビュー（デプロイ済みページ）
2. 一時ディレクトリでの隔離レビューワークフロー:
   - コードベースを見せない状態でエージェントを起動
   - GitHub Pages のデプロイ済み URL にアクセスさせる
   - WebFetch を使って実際のレンダリング結果をレビュー
   - レイアウト、ナビゲーション、リンク切れ、読みやすさを検証

### レビュー観点
- 専門用語の説明が十分か
- 図表が分析をサポートしているか
- 読者が前提知識なしで内容を追えるか
- リンクの遷移が自然か
