# Task 126: コールドスリープの分析制約の明確化

## Status: DONE

## Description
作中でコールドスリープは明示的に言及されている以上、明示的な場面以外ではコールドスリープが
使用されないことを前提に分析を行う。各エピソードの航行時間分析で「コールドスリープを使った
可能性がある」という解釈を安易に適用しないよう、レポートとCLAUDE.mdに制約を追加する。

## Approach
1. 各エピソードレポートでコールドスリープに言及している箇所を確認
2. コールドスリープが明示的に使用されるシーンを特定
3. それ以外の航行中はコールドスリープなしで分析されていることを確認
4. 必要に応じてレポートを修正

## Dependencies
- 各エピソードレポート (ep01-ep05.json)
- ship-kestrel.json

## Origin
人間指示: 「コールドスリープについては、作中で明示的に言及されている以上、明示的な場面以外で行われないはず」
