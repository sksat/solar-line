# Task 123: 数値分析のスクリプト化＋一括更新インフラ

## Status: DONE

## Description
数値分析（ΔV計算、質量予算、軌道パラメータ等）を再現可能なスクリプトとしてまとめ、
レポート内に再現コマンドを記載し、一括更新できるインフラを構築する。

## Approach
1. 既存の分析をスクリプト化（Rust CLI or TypeScript）
   - EP01-05 の ΔV 計算
   - 質量バジェット計算
   - ノズル寿命計算
   - 推進剤予算計算
2. 各スクリプトの出力をレポートデータに反映するパイプライン
3. レポートのコンポーネント内に再現コマンドを記載
4. `npm run recalculate` 的な一括更新コマンドの作成
5. パラメータ変更時の影響範囲をDAGで追跡

## Dependencies
- crates/solar-line-core/ (計算ロジック)
- reports/data/ (レポートデータ)
- dag/ (依存関係追跡)

## Origin
人間指示: 「数値的な分析・解析は、スクリプトとしてまとめておいて、すぐに再現可能にしておくこと。レポートのコンポーネント内などに再現のためのコマンドなども残しておくとよい。また、それらのコマンドをある程度まとめて実行してパラメータ群を一気に更新するインフラがあるとよりロバストな更新ができるようになる。」
